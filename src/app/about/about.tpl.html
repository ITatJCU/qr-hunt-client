Frequently Asked Questions

How is the game ranked?

Each player has a collected number of scans linked to their player id, and a time stamp for each. The server tasked with tracking all players compares the total number of codes collected and a duration of time between the first and last code scanned.

The players with the most scans in the shortest period of time are ranked highest.

What if I scan the same code twice?

In most cases, nothing, but the game keeps track of the time between the first and last unique scans, so rescanning the first or last code will shorten or lengthen your time respectively.

Find all the codes and race between them to reduce your time!

Why don't I need an account?

The Web client generates a universally unique identifier (UUID) and stores it locally on the device using a cookie. Every time the application is reloaded, the cookie containing the UUID is restored and associated player information is retrieved.

This technique is often used by mobile advertisers to generate targeted advertisements.

Is this a student project?

We came up with a rough idea in a brainstorming session, then a motivated student volunteered to implement QRHunt as a pet project.

Developer-Related Questions

What languages and frameworks are used?

The Client

The client makes use of a relatively new Web-oriented framework called AngularJS by leveraging the popular ngBoilerplate build management system in combination with the mobile-first front-end framework, Twitter Bootstrap.

The Server

The server is written in JavaScript for the synchronous event-driven framework Node.js and currently running on a Raspberry Pi (mini computer) somewhere in Townsville.

Development time was reduced by utilising the restify and Sequelize modules for Node.js.

Is the source code available?

All source code for the client and server are available on GitHub at: QR Hunt Client and QR Hunt Server respectively.

What about the QR codes?

QR codes have been around since 1994, and are a very effective way of encoding information. You can generate your own QR code for a URL using free websites like QR-Code-Monkey.

Each of our QR Codes points to a unique URL in the QRHunt game, either an information page like this, or a specific code that you need to collect.
